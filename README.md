# USA Phone Number Validator

A pure client-side web application that validates, cleans, and organizes USA phone numbers by state. 100% runs in your browser - no server required!

## Features

- ✅ **Bulk Processing**: Upload .txt files with up to 10,000 numbers
- ✅ **Automatic Cleaning**: Removes +1 prefix and non-numeric characters
- ✅ **State Validation**: Validates against all 50 USA states area codes
- ✅ **State-wise Organization**: Groups numbers by state
- ✅ **Multiple Downloads**: Download results as individual state files or ZIP
- ✅ **100% Client-side**: All processing happens in your browser
- ✅ **No Data Sent**: Your data never leaves your computer
- ✅ **Drag & Drop**: Easy file upload interface
- ✅ **Progress Indicator**: Real-time processing progress

## How to Use

1. **Visit the website**: Open `index.html` in your browser
2. **Prepare your file**: Create a .txt file with one phone number per line
3. **Upload**: Drag & drop or click to upload your file
4. **Process**: Click "Process Numbers"
5. **Download**: Get cleaned numbers organized by state
